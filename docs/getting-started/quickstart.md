# Quickstart

# Installation
### Requirements
* **Python**: 3.12 or higher

### Installing the Library

`pqlattice` is available on PyPI - https://pypi.org/project/pqlattice/

=== "pip"
    ```bash
    pip install pqlattice
    ```
=== "uv"
    ```bash
    uv add pqlattice
    ```

# Basis usage

```python
import pqlattice as pq
```
